# 🚗 Front-End - Sistema de Locadora de Veículos

Este repositório contém o front-end do sistema de uma locadora de veículos, responsável por apresentar a interface visual do site, onde usuários podem visualizar veículos, realizar reservas e navegar pelas funcionalidades da aplicação.

# 🖥️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript 
* Bootstrap 5

# 🎨 Funcionalidades Visuais

* 🏠 Página de login para admin e user 
* 🚗 Listagem de veículos disponíveis para locação
* 📝 Formulário de adicionar veiculo e outro de cauculo de aluguel
* 📱 Layout responsivo para dispositivos móveis
